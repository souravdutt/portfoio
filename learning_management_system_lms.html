Learning Management System

Description

It is not a readymade Wordpress LMS. It is created by me from the scratch using Vanilla PHP and SQL.
The skill set used for creating this Learning Management Sysytem (LMS) is PHP, MySQL, HTML5, CSS3, Bootstrap, JQuery and Ajax. Which is designed and developed by Soruav Dutt.
The user interface is clean and completely responsive. Below is a quick introductive video for this Learning Management System. If you need a demo to test this LMS then click here to contact me.

Featurs

→ Any kind of Institute can use this LMS either it is an acadamy/institute or huge college/university.
→ Three user interfaces for Admin, Teacher and Students.
→ Admin can create courses/classes and subjects for each course.
→ Teacher can upload Video or PDF or Image file only for the course and subjects that are alloted to him/her.
→ Students can access study material only for his/her course.
→ Students can discuss about a topic with other classmates and teachers in comments on a video.
→ Students can submit assignments/homework as PDF or Image File and teacher can give them scores.
→ Admin can Deactivate/Activate any student or teacher any time if needed.

There are lots of more features, Contact Me to discuss and to know more about this LMS Project.
